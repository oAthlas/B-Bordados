# B-Bordados E-commerce Platform

## Visão Geral do Projeto
O B-Bordados é uma plataforma de e-commerce dedicada à comercialização de bordados personalizados, oferecendo uma experiência única e intuitiva para os clientes.

## Recursos
- Catálogo de produtos variados.
- Personalização de pedidos.
- Carrinho de compras otimizado.
- Sistema de pagamento seguro.
- Perfil de usuário com histórico de compras.

## Tecnologias
- Backend: Django
- Banco de dados: PostgreSQL (Supabase)
- Integrações: APIs para pagamento.

## Estrutura do Site
A estrutura do site é organizada em:
- Página Inicial
- Galeria de Produtos
- Página de Produto
- Login/Criação de Conta
- Carrinho de Compras
- Checkout

## Modelos de Dados
Os principais modelos de dados utilizados incluem:
- Usuário
- Produto
- Pedido
- Pagamento

## Segurança
Implementamos diversas medidas de segurança, incluindo:
- Criptografia de dados sensíveis.
- Verificações de autenticação.
- Proteção contra ataques de injeção de SQL.

## Integrações
A plataforma é integrada com:
- Provedores de pagamento (Mercado Pago).

## Jornada do Cliente
A jornada do cliente na plataforma inclui:
1. Navegação pelo catálogo de produtos.
2. Personalização do pedido.
3. Adição ao carrinho de compras.
4. Finalização da compra com pagamento seguro.
5. Recebimento de confirmações e atualizações sobre o status do pedido.
